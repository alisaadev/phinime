# Phinime

A mobile streaming application for anime built with Expo and React Native.

## 📱 About

Phinime is a TypeScript-based Expo React Native application that provides a streaming platform for anime content. The app leverages the Sankavolerrai API to deliver a seamless anime watching experience on mobile devices.

## 🛠️ Tech Stack

- **Framework**: Expo + React Native
- **Language**: TypeScript (99.8%), JavaScript (0.2%)
- **API**: Sankavolerrai

## ✨ Features

- Stream anime content on mobile devices
- Built with Expo for cross-platform compatibility
- Powered by Sankavolerrai API

## 🚀 Getting Started

To get started with this project, clone the repository and install the dependencies:

```bash
git clone https://github.com/alisanooralya/phinime.git
cd phinime
npm install
# or
yarn install
```

Run the application:

```bash
npx expo start
```
